# KB Dashboard — Web UI

Real-time web dashboard for KB + AADS monitoring and control.
Built with React, TypeScript, and D3.js.

## Screens (from Figma)
- Overview              — System health, active alerts summary
- Swarm Topology        — D3.js force graph of agent network
- Anomalies & Alerts    — Real-time alert management
- Role Distribution     — Agent role breakdown + trends
- Swarm Control         — Manage swarm deployments
- Agent Lifecycle       — Per-agent management
- Quorum & Consensus    — Active voting dashboard
- Lifecycle Monitoring  — KPI tracking
- Rogue Management      — Sandboxed/flagged agents
- Signal Relay          — P2P communication matrix
- Shared Mind           — Collective knowledge graph
- Honeypot Management   — Deceptive node management
- Governance            — Behavioral judiciary
- System Monitor        — Event bus + model sync
- Reports & Audit       — Immutable audit trail
- Settings              — RBAC, API tokens, config

## Run
```bash
npm install
npm run dev
```
